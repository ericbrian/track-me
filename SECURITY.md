# Security Policy

## Overview

TrackMe is a privacy-first iOS application for GPS tracking and session management. Security and user privacy are fundamental to the app's design.

## Supported Versions

TrackMe is currently in active development. Security updates will be provided for:

| Version | Supported          | Status |
| ------- | ------------------ | ------ |
| main    | :white_check_mark: | Active development |
| 1.0.x   | :white_check_mark: | Planned first release |

## Reporting a Vulnerability

If you discover a security vulnerability in TrackMe, please report it responsibly:

### How to Report

1. **Email**: Send details to the repository owner via GitHub
2. **GitHub Security Advisory**: Use GitHub's private security advisory feature
3. **Do Not**: Create public issues for security vulnerabilities

### What to Include

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact assessment
- Suggested fix (if available)

## Contact

For security concerns or questions, contact the maintainers through:

- GitHub Issues (for non-sensitive matters)
- GitHub Security Advisories (for vulnerabilities)
- Repository owner contact (available in GitHub profile)
